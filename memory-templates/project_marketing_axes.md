{{BRAND}}'s Marketing AI System calibrates every script on two axes, not one:

- **Funnel Layer (1-5)** — content distribution mix. Decides CTA behavior and whether {{BRAND}} is mentioned. Defined in `{{VAULT_PATH}}/social-media/Content Guide.md`.
- **Awareness Level (A-E, Schwartz)** — viewer mental state. A=Unaware, B=Problem-Aware, C=Solution-Aware, D=Product-Aware, E=Most Aware. Decides hook technique and body framing.

**Why:** Christian uses both in his Notion Content Delivery DB as separate metadata fields on each idea. Before 2026-04-19 the vault docs only formalized Funnel Layer — Awareness Level was in his head and in Notion but invisible to Layer 1 generators. Added as a first-class dimension in `Marketing AI System.md` and `Content Guide.md` on 2026-04-19.

**How to apply:** when writing, reviewing, or calibrating a {{BRAND}} script, pull both dimensions. Funnel Layer 1 + Awareness A ≠ Funnel Layer 1 + Awareness B — same distribution slot, different hook and body framing. If only one axis is given, infer the other using the default mapping in `Marketing AI System.md` → "Awareness level calibration" (e.g. Layer 1 defaults to Awareness A-B, Layer 3-4 defaults to C-D, Layer 5 defaults to D-E).
